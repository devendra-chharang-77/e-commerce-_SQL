E-Commerce Data Analysis using MySQL + Python

📌 Project Overview:
This project focuses on performing SQL-based data analysis on an e-commerce dataset.
Data is imported from CSV files into MySQL using Python, and multiple SQL queries are executed to extract insights.

Dataset
Source: Kaggle
Dataset: E-Commerce Target Sales Dataset
Link: https://www.kaggle.com/datasets/narendrabariha/e-commerce-target-sales-dataset

🛠️ Tech Stack
Python (Pandas, MySQL Connector)
MySQL
Jupyter Notebook
GitHub
